docker-reactor
==============

Personal Docker / React playground.

### Run from the build:

- `docker run -p 8888:8888 nicksergeant/docker-react`

### Run it locally:

1. `git clone https://github.com/nicksergeant/docker-reactor.git`
2. `cd docker-reacter`
3. `node server`

### Compile it:

- `make bundle`

### Watch it:

1. `npm install -g watch`
2. `watch 'make bundle' static/app`
